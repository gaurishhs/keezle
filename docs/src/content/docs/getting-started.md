## Installation

```bash
go get github.com/gaurishhs/keezle
```

## Usage

Import the package and create a new instance of Keezle specifying the configuration.

```go
package main

import (
	"github.com/gaurishhs/keezle"
)

func main() {
	keezle := keezle.New()
}
```
